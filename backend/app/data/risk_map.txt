# [SafeHomeAI 안전 진단 가이드라인: 발달 단계 중심]

이 문서는 YOLO 탐지 객체와 아동의 발달 단계를 매핑하여 위험도를 판단하는 절대적인 규칙입니다. 탐지된 객체가 아래 리스트에 포함되어 있다면 반드시 해당 위험도(Danger/Warning)를 할당해야 합니다.

## 1. 위험도 등급 정의 (Risk Levels)
* **🚨 위험 (Danger)**: 생명 위협(질식, 추락, 익사, 감전, 중독) 또는 영구적 손상/중상. (즉각 조치 필수)
* **⚠️ 경고 (Warning)**: 경미한 부상(타박상, 찰과상) 또는 환경 개선이 필요한 잠재적 위험.

---

## 2. 발달 단계별 위험 객체 매핑 (Stage-Specific Rules)

### 📌 단계 1: 누워있는 시기 (The Lying Down Stage)
* **참고 연령:** 0~3개월
* **핵심 특성:** 이동 불가, 고개 가누기, 쥐기 반사.
* **[Pillow, Stuffed Toy, Couch]** -> **🚨 위험** (질식: 코와 입 막힘)
* **[Bed]** (가드 없음) -> **⚠️ 경고** (낙상: 뒤척임)
* **[Monitor/TV, Potted Plant, Clock, Picture/Frame]** (머리맡) -> **⚠️ 경고** (낙하: 아기 위로 떨어짐)

### 📌 단계 2: 앉거나 기기 시작하는 시기 (The Sitting & Crawling Stage)
* **참고 연령:** 4~6개월
* **핵심 특성:** 뒤집기, 배밀이, 손 뻗기, 구강기(입으로 가져감).
* **[Bed, Couch]** -> **🚨 위험** (추락: 뒤집기로 떨어짐)
* **[Kettle, Rice Cooker, Pot]** -> **🚨 위험** (화상: 뜨거운 본체 터치)
* **[Key, Eraser, Lipstick, Coin]** -> **🚨 위험** (질식: 직경 3.5cm 이하 삼킴)
* **[Fan]** -> **⚠️ 경고** (끼임: 날개에 손가락 넣음)

### 📌 단계 3: 보행기 및 탐색기 (The Walker & Explorer Stage)
* **참고 연령:** 7~12개월
* **핵심 특성:** 잡고 서기, 서랍 열기, 식탁보 당기기.
* **[Power outlet]** -> **🚨 위험** (감전: 이물질 삽입)
* **[Dining Table]** -> **🚨 위험** (낙하/화상: 식탁보 당김)
* **[Bathtub, Washing Machine/Drying Machine, Toilet]** -> **🚨 위험** (익사/갇힘: 5cm 물에도 익사 가능)
* **[Cabinet/shelf, TV]** (고정되지 않은 경우) -> **⚠️ 경고** (전도: 잡고 서다 넘어짐)
* **[Trash bin Can]** -> **🚨 위험** (오염/삼킴: 쓰레기 뒤짐)

### 📌 단계 4: 혼자 걷기 시작하는 시기 (The Walking Alone Stage)
* **참고 연령:** 1~3세 (※ 디폴트 설정 구간)
* **핵심 특성:** 활동 반경 확대, 모방 행동, 전선 걸림, 가구 등반.
* **[Gas stove, Induction Cooker]** -> **🚨 위험** (화재/화상: 밸브/터치 조작)
* **[Cleaning Products, Cosmetics, Soap]** -> **🚨 위험** (중독: 음료/사탕으로 오인 섭취)
* **[Window]** -> **🚨 위험** (추락: 난간 접근)
* **[Fork, Chopsticks, Pen/Pencil, Toothbrush]** -> **🚨 위험** (찔림: 입에 물고 뛰다 넘어짐)
* **[Door]** -> **⚠️ 경고** (끼임: 손가락 절단/골절 위험)
* **[Router/modem, Laptop, Mouse]** (전선) -> **⚠️ 경고** (걸림: 전선에 걸려 넘어짐)
* **[Dining Table, Table, Coffee Table]** -> **⚠️ 경고** (충돌: 눈높이 모서리에 머리 찧음)
* **[Chair, Stool]** -> **⚠️ 경고** (낙상/전도: 밟고 올라가거나 의자가 넘어짐)
* **[Cabinet/shelf]** (고정 안 된 서랍장) -> **⚠️ 경고** (끼임/전도: 서랍장 열기 및 매달리기)

### 📌 단계 5: 활동 범위가 확대되는 시기 (The Activity Expansion Stage)
* **참고 연령:** 3~5세
* **핵심 특성:** 높은 곳 오르기, 도구 사용 시도, 불 호기심.
* **[Chair, Stool, Ladder, Box]** (창가/난간 옆) -> **🚨 위험** (추락 보조물: 딛고 올라감)
* **[Lighter, Candle]** -> **🚨 위험** (화재: 불장난)
* **[Blender, Microwave, Toaster, Coffee Machine]** -> **🚨 위험** (베임/화상: 기기 작동)
* **[Iron]** -> **⚠️ 경고** (화상: 잔열 접촉)
* **[Mop, Broom]** -> **⚠️ 경고** (넘어짐: 긴 자루에 걸림)

### 📌 단계 6: 독립성이 발달하는 시기 (The Independence Stage)
* **참고 연령:** 5세 이상
* **핵심 특성:** 대근육 활동, 부주의.
* **[Treadmill]** -> **⚠️ 경고** (마찰화상/끼임: 작동 중 장난)
* **[Bicycle]** (실내) -> **⚠️ 경고** (충돌: 실내 주행)

## 3. 공통 적용 절대 규칙 (Universal Absolute Rules)
**※ 아래 객체는 발달 단계와 상관없이 탐지되는 즉시 해당 위험도를 부여하십시오.**

1.  **위험한 공구 및 도구 (Dangerous Tools)**
    * **대상:** `Hammer`, `Electric Drill`, `Screwdriver`, `Pliers`, `Saw`, `Knife`, `Scissors`
    * **판정:** **🚨 위험 (Danger)**
    * **이유:** 날카롭거나 무거워 아이가 만질 경우 심각한 물리적 부상 유발.

2.  **고위험 발열 전자기기 (High-Heat Appliances)**
    * **대상:** `Hair Dryer`, `Radiator`, `Iron`, `Toaster`, `Kettle`, `Oven`
    * **판정:** **🚨 위험 (Danger)**
    * **이유:** 전원이 꺼져 있어도 잔열로 인한 화상 및 감전 위험.

3.  **깨지기 쉬운 물건 (Breakables)**
    * **대상:** `Wine Glass`, `Vase`, `Mirror` (벽걸이 제외, 세워둔 전신거울 한정)
    * **판정:** **🚨 위험 (Danger)**
    * **이유:** 파손 시 유리 파편에 의한 깊은 베임 사고 위험.

4.  **삼킴 위험 (Choking Hazard) - [0~3세 한정]**
    * **대상:** `Key`, `Coin`, `Lipstick`, `Eraser`, `Battery`, `Remote`, `Wallet/Purse`
    * **판정:** **🚨 위험 (Danger)**
    * **이유:** 구강기 아동의 기도 폐쇄 유발. (리모컨: 건전지, 지갑: 동전 포함)
